About

VbVt is a vanilla anarchy server with the goal to stay true to the vanilla survival gamemode. Feel free to PVP, grief, build, and say whatever you want. The only rule is to not use hacked clients.

The version to join the server is 1.20.1, The Trails & Tales Update. To make sure message are safe from reporting we disabled Microsoft reporting. The ip to connect is mc.VbVt.ml

Server Rules

DO NOT create lag machines. anything that can intentionally break or use up server resources

DO NOT use hack clients. hacked clients, external programs, x-ray resource packs

DO NOT use exploits. any sort of duping

Need help?

If you have any questions or issues join the Discord Server and make a support ticket in #support.
